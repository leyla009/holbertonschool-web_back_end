# Python Variable Annotations

This directory contains exercises and examples focused on **Type Hinting** in Python. Type annotations allow for better code readability, improved tooling support (like IDE autocompletion and static analysis), and clearer documentation of expected data types for function arguments and return values.

## Exercises

| File | Description |
| :--- | :--- |
| `0-add.py` | A type-annotated function `add` that returns the sum of two floats. |

## Requirements
* All scripts are written in Python 3.
* All scripts follow the PEP 8 style guide.
* All scripts must start with the `#!/usr/bin/python3` shebang.
